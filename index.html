🤖 วิเคราะห์ Training Load จาก Strava
วิธีรับ Refresh Token (ใช้ครั้งเดียว — ไม่หมดอายุ)
  1. ไปที่ strava.com/settings/api — ดู Client ID และ Client Secret แล้วบันทึกในหน้าตั้งค่าก่อน
  2. เปิด URL นี้ใน browser (แทน CLIENT_ID):
    https://www.strava.com/oauth/authorize?client_id=CLIENT_ID&redirect_uri=http://localhost&response_type=code&scope=activity:read_all
  3. กด Authorize → คัดลอก code=xxxx จาก URL
  4. แลก code:
    curl -X POST https://www.strava.com/oauth/token -d "client_id=ID&client_secret=SECRET&code=CODE&grant_type=authorization_code"
  5. คัดลอก refresh_token แล้วกด "เชื่อมต่อ" ด้านบน ✅
