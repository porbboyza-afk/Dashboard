How to connect Strava
  1. ไปที่ strava.com/settings/api — ดู Client ID และ Client Secret แล้วบันทึกในหน้า Settings ก่อน
  2. ตั้ง Redirect URI ให้ตรงกับ Callback Domain ใน Strava app. ใช้ http://localhost ได้ถ้าตั้งไว้แบบเดิม
  3. กด Connect แล้วเลือก Open Strava auth. MyDash จะขอ scope activity:read,activity:read_all
  4. กด Authorize แล้วคัดลอก code=xxxx จาก URL กลับมาใส่ช่อง Connect code
  5. ถ้าเจอ 403 ตอน sync ให้ Disconnect แล้วเชื่อมใหม่ด้วยวิธีนี้ เพื่อให้ token มี scope อ่านกิจกรรมครบ
